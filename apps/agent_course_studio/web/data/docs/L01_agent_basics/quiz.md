# L01 课后小测（讲义版）

请先独立作答，再看参考答案。

1. 用一句话定义 AI Agent。
2. ChatBot 与 Agent 的核心差异是什么？
3. Agent 的四个核心模块分别是什么？
4. 为什么说 Tools 是 Agent 的“手脚”？
5. 在 Agent 流程中，Planning 的价值是什么？
6. 多轮对话里，`messages` 列表为什么能形成“最小记忆”？
7. 举一个“只用 Prompt 就够”的任务。
8. 举一个“必须用 Agent 更合适”的任务。
9. 说出两个常见失败模式。
10. 你当前工作里，有什么任务可以改造成 Agent 流程？至少写 2 个步骤。

## 参考答案（示例）

1. AI Agent 是以 LLM 为核心，具备规划、工具调用和记忆能力的智能程序。
2. ChatBot 主要回答问题；Agent 以“完成任务”为目标并可自主执行。
3. LLM、Planning、Tools、Memory。
4. 因为它能连接外部系统并执行真实动作，而不仅是文本回答。
5. 把复杂任务拆成可执行的小步骤，降低一次性完成的难度和失败率。
6. 每轮的用户与助手消息都被保存，下一轮请求时模型可看到完整上下文。
7. 文案润色、标题改写、语法纠错等单步任务。
8. “搜集资料 -> 对比分析 -> 生成报告 -> 邮件发送”这类多步骤任务。
9. 幻觉、工具调用失败、循环不收敛、上下文超限等。
10. 示例：周报自动生成。步骤：抓取数据 -> 分析趋势 -> 生成草稿 -> 人工审核。
